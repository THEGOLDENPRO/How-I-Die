# *Set Up*
**1) Download git repo.**
```
git clone https://github.com/THEGOLDENPRO/How-I-Die
```

**2) Install all required dependencies.**
```
pip install -r requirements.txt
```

**3) Run app.**
```
python how_i_die.py
```
